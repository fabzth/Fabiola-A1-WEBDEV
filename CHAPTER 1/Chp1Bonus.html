HTML & CSS Quick Reference Guide

An overview of essential HTML tags and CSS properties

HTML Tags

<h1> to <h6>

Defines HTML headings. <h1> is the largest, and <h6> is the smallest.

<h1>This is a Heading 1</h1>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

<a>

Defines a hyperlink. Use the href attribute to define the link's destination.

<a href="https://example.com">This is a link</a>

<img>

Embeds an image. Use the src attribute to specify the image source.

<img src="image.jpg" alt="Image Description">

<ul> and <li>

Defines an unordered list with list items.

<ul>
    <li>Item 1</li> 
    <li>Item 2</li> 
</ul>

<ol> and <li>

Defines an ordered list.

<ol>
    <li>First item</li> 
    <li>Second item</li> 
</ol>

<div>

Defines a division or section in an HTML document.

<div>This is a division</div>

<span>

Defines a section in a document. It is used as an inline container.

<span>This is a span</span>

<table>

Creates a table. Use <tr> for rows, <td> for data cells, and <th> for headers.

<table>
    <tr> 
        <th>Header 1</th> 
        <th>Header 2</th> 
    </tr> 
    <tr> 
        <td>Data 1</td> 
        <td>Data 2</td> 
    </tr> 
</table>

<form>

Defines an HTML form for user input.

<form action="submit.php" method="post"> 
    <input type="text" name="name"> 
    <input type="submit" value="Submit"> 
</form>

CSS Properties

color

Sets the color of text.

p {
    color: blue; 
}

background-color

Sets the background color of an element.

div {
    background-color: #f0f0f0; 
}

font-size

Sets the size of the font.

h1 {
    font-size: 24px; 
}

margin

Sets the margin space around an element.

p {
    margin: 10px; 
}

padding

Sets the padding space inside an element.

div {
    padding: 15px; 
}

border

Sets border around an element.

img {
    border: 2px solid black; 
}

text-align

Sets the horizontal alignment of text.

h2 {
    text-align: center; 
}

display

Specifies the display behavior of an element.

span {
    display: inline-block; 
}
